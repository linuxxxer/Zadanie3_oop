Funkcionality:
    -> Open file                        - Handles openButton
    -> Save file                        - Handles saveButton
    -> Add transition                   - Handles addTransitionButton
    -> Add place                        - Handles addPlaceButton
    -> Add arc/reset arc                - Handles addArcButton / addResetArcButton
    -> Set multiplicity on arcs         - Handles interactButton
    -> Set labels                       - Handles setLabelButton
    -> Set tokens                       - Handles interactButton
    -> Fire transitions                 - Handles interactButton
    -> Move objects                     - Handles reSetPositionButton
    -> Remove objects individually      - Handles removeButton
    -> Clear current Petri Net          - Handles clearButton
    -> CTRL + Q -- closes the program   - Handles KeyAdatper in NetsFrame
    -> CTRL + S -- opens save dialog    - Handles KeyAdatper in NetsFrame
    -> CTRL + O -- opens open dialog    - Handles KeyAdatper in NetsFrame
